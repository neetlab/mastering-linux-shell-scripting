# 覚えたこと

## 記号類

### 2>&1

標準エラーを標準出力に流す。

```bash
# エラーだけが画面に出るようにしたい場合:
ping -c3 $server_addr \
  2>&1 \      # 標準エラーを標準出力に
  > /dev/null # 標準出力を/dev/nullに
```

### && と ||

左辺のプロセスの終了コードを見て、右辺の実行を判断する。

### $#

引数の個数

## コマンド

### type

その名前の型を表示する

### read

インタラクティブにユーザーに質問する

```bash
#!/bin/sh
read -p "名前を入力: " name
echo $name
```

### shift

引数 `$@` をシフトする

### test EXPR （省略記法 [ EXPR ]）

式の真偽を判定して、終了コードに結果を入れる。**真のときにゼロ**。

```
test EXPR
test ! EXPR
```

式の種類はマジでいっぱいある

```
test -z $str1
test -n $str1
test $str1 = $str2
```
